# Nodejs Typescript starter

## Details

This template provide quickstart configuration for Node.js

Includes :
- typescript
- eslint
- prettier
- jest
- husky
- express
- dotenv

## Commands

```bash
#installation
yarn

#run dev
yarn start:dev

#run production
yarn start

#test
yarn test

#lint
yarn lint

#prettier
yarn format

#build
yarn build
```
