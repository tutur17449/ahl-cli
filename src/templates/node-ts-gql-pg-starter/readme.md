# Nodejs Typescript Graphql starter with postgresql

## Details

This template provide quickstart configuration for Node.js

Includes :
- typescript
- eslint
- prettier
- jest
- husky
- express
- pg
- graphql
- @graphql-codegen
- joi
- knex
- dataloader
- apollo-server-express
- dotenv

## Commands

```bash
#installation
yarn

#generate type from graphql schema
yarn gen

#run database
yarn start:db

#run dev
yarn start:dev

#run production
yarn start

#create migration
yarn migration:create <migration_name>

#run migration
yarn migration:run

#rollback migration
yarn migration:rollback

#run seed
yarn seed

#test
yarn test

#lint
yarn lint

#prettier
yarn format

#build
yarn build
```
